# Catalog of My Things <a name="about-project"></a>


**Catalog of My Things** is an app that helps you to keep a record of different types of things you own: books, music albums, movies, and games. 
Everything is based on the UML class diagram presented below. The data is be stored in JSON files but we also prepared a database with tables structure analogical to our program's class structure.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd enumerable
  git clone https://github.com/Jaymelfah/Catalog-of-things.git
```

### Install

Install this project with:

```sh
  gem install
```

### Usage

To run the project, execute the following command:

Example command:

```sh
  ruby my_list.rb
```
